# DW-Assignment2

This website was developed for the "Web Development" course of the Master's in Design and Multimedia of the Faculty of Sciences and Technology of the University of Coimbra.

The main objective was to develop a static webpage using pure HTML, CSS and JavaScript, including animations, interactions and data obtained from any public API.

The website consists of a personal portfolio, featuring projects consisting of single images.

## APIs Used

**Lorem Picsum** <https://picsum.photos/>
Used to retrieve random photos.

**Rando** <https://random-word-api.vercel.app/>
Used to retrieve random words to simulate descriptions.
